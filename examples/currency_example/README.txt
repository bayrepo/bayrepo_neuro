Поиск в закономерности изменения курса валют в зависимости от дня и месяца.

Обучение, на сонове истории изменения курса рубля к доллару:
go run neuro.go valut2.xml 2
где valut2.xml файл с курсом, исторический и 2 - номер сетки в ферме для обучения

go run neuro3.go
запустится веб-сервер, где будет отображен график изменения курса и как сетка спрогнозировала падения и нарастания курса, в зависимости от дня и месяца года.

Фактически закономерности нет :(
